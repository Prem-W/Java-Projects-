# Java-Projects-
1. Hospital Management:-
Built a hospital management solution to handle patient and doctor information, 
appointments, and billing. Solves the problem of manual record maintenance by improving data organization and retrieval. Key features include patient registration, doctor management, appointment scheduling, bill generation, and record search. Contributed to system design, core logic implementation, and testing of all modules to ensure smooth and error-free workflow automation.
2. GuesserGameProject:-
Developed a fun logic-based game where the system generates a random number and the player attempts to guess it within limited tries. Solves the problem of user-engagement and logic implementation learning through interactive gameplay. Key features include random number generation, attempt counter, hint feedback (too high/low), input validation, and replay option. Contributed to game logic design, user input handling, loop structuring, and testing to ensure a smooth and error-free gaming experience.
